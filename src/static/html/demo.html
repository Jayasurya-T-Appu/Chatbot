Quick Integration

Add the chatbot to your website in just 2 steps:

<!-- Step 1: Add the widget script -->
<script src="https://your-domain.com/widget.js"></script>

<!-- Step 2: Initialize with your settings -->
<script>
  ChatBotWidget.init({
    clientId: 'your-client-id',
    apiKey: 'your-api-key',
    title: 'Chat with us',
    primaryColor: '#007bff'
  });
</script>

Configuration Options:

  • clientId (required): Your unique client identifier
  • apiKey (required): Your API authentication key
  • title: Chat window title (default: "Chat with us")
  • primaryColor: Main color theme (default: "#007bff")
  • position: Widget position (default: "bottom-right")
  • welcomeMessage: Initial greeting message
  • apiUrl: API endpoint URL
